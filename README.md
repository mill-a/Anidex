# Anidex
* Käyttäjät pystyvät lisäämään sovellukseen löytämiään eläimiä (myös lemmikkejään). Samantapainen kuin Pokédex.
* Käyttäjä pystyy luomaan tunnuksen ja kirjautumaan sisään sovellukseen.
* Käyttäjä pystyy lisäämään, muokkaamaan ja poistamaan eläimiä.
* Käyttäjä pystyy lisäämään kuvia eläimestä.
* Käyttäjä näkee sovellukseen lisätyt eläimet.
* Käyttäjä pystyy etsimään eläimiä hakusanalla.
* Sovelluksessa on käyttäjäsivut, jotka näyttävät tilastoja ja käyttäjän lisäämät eläimet.
* Käyttäjä pystyy valitsemaan eläimelle yhden tai useamman luokittelun (esim. laji, naaras/uros, ikä).
* Käyttäjä pystyy arvostelemaan ja kommentoimaan muiden eläimiä.
